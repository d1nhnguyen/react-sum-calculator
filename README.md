# React Sum Calculator

A simple React application that allows users to input two numbers and display their sum dynamically.

##  Features
- Two input boxes for numbers
- "Calculate Sum" button
- Validation for empty or invalid input
- Dynamic result display

##  Built With
- React Functional Components
- React Hooks (`useState`)
- Event Handling (`onChange`, `onClick`)
- Styled with plain CSS

## How to Run Locally
1. Clone this repository
2. Run `npm install`
3. Start the project:
   ```bash
   npm start
